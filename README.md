# GoRead – Книжный клуб

## Тема проекта
Сайт для книжного клуба и сообщества читателей.

## Описание проекта
GoRead — это веб-сайт книжного клуба, созданный для людей, которые любят читать и обсуждать книги.  
Платформа объединяет читателей, помогает делиться мнениями, узнавать о мероприятиях и находить интересные статьи о литературе.

## Страницы сайта
- Главная (index.html)  
- О клубе (about.html)  
- Клуб (club.html)  
- Статьи (articles.html)  
- Мероприятия (events.html)

## Используемые технологии
- HTML5  
- CSS3
